## <img src="https://img.icons8.com/fluency-systems-regular/24/ffffff/overview-pages-3.png"/> Overview

**OptiCore** is an interactive system that uses **Genetic Algorithms (GA)** to optimize CPU scheduling.  
It benchmarks and evolves hybrids of classic algorithms such as **FCFS**, **SJF**, **Priority**, and **Round Robin** to minimize:
- Average Waiting Time  
- Turnaround Time  
- Context Switches  
- Fairness Deviation  

---

## <img src="https://img.icons8.com/fluency/24/rocket.png"/> Features

- Compare multiple baseline algorithms  
- Evolve optimized schedules using GA  
- Interactive sliders for workload and GA tuning  
- Visualize with **Gantt charts** and **performance graphs**  
- Export or display metrics dynamically in Colab

## <img src="https://img.icons8.com/external-flat-icons-inmotus-design/24/ffffff/external-code-automation-flat-icons-inmotus-design.png"/> How to Run

1. Open the OptiCore notebook in **Google Colab**.  
2. Run all cells in order.  
3. Adjust the sliders for:
   - Process count  
   - Population size  
   - Generations  
   - Mutation rate  
4. Click **Run GA Optimizer** to evolve an optimal schedule.  
5. Click **Compare Baselines** to benchmark traditional algorithms.
